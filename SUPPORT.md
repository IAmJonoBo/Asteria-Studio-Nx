# Support

## Where to Get Help

- **Bug reports**: Use the GitHub issue templates in this repo.
- **Feature requests**: Use the feature request template.
- **Questions and discussion**: Use GitHub Discussions if enabled; otherwise open an issue.

## What to Include

Please include:

- Version/commit SHA
- OS and Node.js version (`node -v`)
- Steps to reproduce
- Logs or screenshots (redact sensitive data)

## Security

For security issues, do **not** open a public issue. Follow `SECURITY.md` instead.
